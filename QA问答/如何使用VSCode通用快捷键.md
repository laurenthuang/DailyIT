## 通用

| 类别  | 功能                | 键             |
| --- | ----------------- | ------------- |
| 视图  | 显示/隐藏主侧边栏         | `Ctrl` + `B`        |
| 视图  | 显示/隐藏副侧边栏         | `Ctrl` + `Alt`+ `B`    |
| 视图  | 显示Command Palette | `Ctrl` + `Shift` + `P`  |
| 视图  | 显示Shortcuts       | `Ctrl` + `K` `Ctrl` + `S` |
| 视图  | 显示User Settings   | `Ctrl` + `,`        |
| 视图  | 显示/隐藏Terminal     | `Ctrl` + `      |
