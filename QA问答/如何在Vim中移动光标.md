## 移动光标

| 类别  | 功能            | 键      |     |
| --- | ------------- | ------ | --- |
| 行移动 | 光标跳到首行首栏      | gg     |     |
| 行移动 | 光标跳到末行首字符     | G      |     |
| 行移动 | 光标跳到\<n\>行首字符 | \<n\>G |     |
| 行移动 | 光标上移\<n\>行    | \<n\>k |     |
| 行移动 | 光标下移\<n\>行    | \<n\>j |     |
|     |               |        |     |
| 栏移动 | 光标左移\<n\>栏    | \<n\>h |     |
| 栏移动 | 光标右移\<n\>栏    | \<n\>l |     |
