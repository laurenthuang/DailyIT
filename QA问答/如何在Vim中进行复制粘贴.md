### 复制粘贴

| 类别   | 功能                               | 键       |
| ---- | -------------------------------- | ------- |
| 复制   | 复制当前行                            | yy      |
| 粘贴   | 粘贴到光标后方                          | p       |
| 编辑   | 不离开Insert Mode输入一次Normal Mode的命令 | Ctrl+o  |
| k寄存器 | 系统剪贴板作为寄存器                       | "+ 或 "* |
