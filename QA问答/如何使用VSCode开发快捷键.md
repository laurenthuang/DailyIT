## 开发
| 类别     | 功能                    | 键                |
| ------ | --------------------- | ---------------- |
| 折叠显示代码 | Fold/Collapse当前区域     | `Ctrl` + `Shift` + `[`   |
| 折叠显示代码 | Unfold/Uncollapse当前区域 | `Ctrl` + `Shift` + `]`   |
| 查询代码   | 去往定义                  | `F12`              |
| 查询代码   | 显示引用                  | `Shift` + `F2`         |
| 查询代码   | Peek定义                | `Alt` + `F12`          |
| 查询代码   | Quick Fix             | `Ctrl` + `.`           |
| 查询代码   | 显示参数提示                | `Ctrl` + `Shift` + `Space` |
